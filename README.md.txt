# Threat Checker GUI

A Python application to detect malicious IP connections using the AbuseIPDB API.

![Screenshot](images/screenshot.png)

## Features
- GUI interface for easy use
- Real-time IP threat checking
- CSV report generation
- Cross-platform support

## Requirements
- Python 3.6+
- [AbuseIPDB API key](https://www.abuseipdb.com/api)

## Installation
```bash
git clone https://github.com/YOUR_USERNAME/threat-checker-gui.git
cd threat-checker-gui
pip install -r requirements.txt
```

## Usage
```bash
python threat_checker.py
```
